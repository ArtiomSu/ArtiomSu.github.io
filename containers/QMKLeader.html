Leader Key Sequences

If you are familiar with how vims leader key works this is pretty much the same. If not then this essentially allows you to program different actions after a certain sequence, you must start the sequence by pressing the key first.

For example if I press the leader key then press h this could type out "hello". If I press leader key then press h and then press e this could type out "hehehehehe". And so on qmk allows your sequences to be up to 5 keys deep.


So lets take an example from the table bellow. For my layout if I press the leader key then I press t, this will send win+enter to open a terminal, it will wait for 200 milliseconds ( this is so that the terminal window has time to open before we continue ) and lastly it will type out "timer"

Another example from the table is if I press the leader key then I press b then I press i, this will type out "https://www.bing.com/images/search?q=" and then I can manually type what I need to search for

Hopefully those two examples gave you an idea of what the leader key does and how the table below works.

Sequence Output
t spawn terminal and type timer
b i type https://www.bing.com/images/search?q=
t o type 10.0.0.69:19009 press enter and type username
h type hahahahahahahahahahaha
c type code . and press enter
right arrow wake leds
right arrow x 2 toggle leds on and off
right arrow x 3 reset all config including hue, brightness, leds etc.... ( will be like when you just plugged in the keyboard )
e type @t80search_bot ( my telegram bot )
Up arrow Go to gaming layer
