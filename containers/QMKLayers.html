Here is my layers visualised for my Ergodox Ez keyboard.

The source code can be viewed at the github link above if you want to see the specific keycodes used.

Layers in qmk are like having different layouts on the same keyboard but you can switch between them by either holding keys, pressing keys.

Layer 1 ( Main Layer )

This is the main layer that you would use to type letters and number. From here you can switch to layer 2 by holding the "hold layer 1, press backspace" button. And this layer has access to the Leader key by pressing leader button on the bottom right. You can also access layer 2 by holding the "hold layer 2, press Enter" key.

Layer 2 ( Symbols Layer )

Layer 2 is essentially everything you get by pressing shift and then sum more commonly used characters + the function keys. What I like about this layer is the way I organised the brackets they are in an easy to remember and easy to reach area. There is also copy and paste from terminal at the top, its easier to press those than reaching for the shift key.

Layer 4 ( Gaming Layer )

The gaming layout shifts the layout by 1 space this is so that wasd keys are in a more triangular shape. This layout also has a full num pad as it is required by some games.
