Source Code

Compile

Git clone the repo:

git clone https://github.com/ArtiomSu/qmk_firmware.git

Switch to my branch

git checkout artiom

compile

make clean && make ergodox_ez:artiom_no_mouse

Or if you want to compile the layout with the custom animations run

make clean && make ergodox_ez:artiom_no_mouse_leds

Now you can flash firmware with teensy flasher.
