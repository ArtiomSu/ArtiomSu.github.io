Features

  • Change leds brightness up and down
  • Change hue up and down
  • Toggle main layer leds
  • Toggle all layer leds
  • Timeout leds after 5 minutes or whatever you set in the config
  • Toggle bunnyhop when space is held
  • Key led notification when leader key is running or when bunnyhop is enabled the = key will blink red.
  • Uses four layers
  • Uses leader key, see leader key table
  • Uses per layer led layouts
  • A lot of different macros for vim mostly
  • Custom animation can be seen in my artiom_no_mouse_leds keymap ( I don't use it with my regular keymap since I don't really need it and it adds a lot of extra code )
